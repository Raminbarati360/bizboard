# وایرفریم متنی صفحات اصلی MVP

## 1. صفحه اصلی AI-first

```
[Header: Logo | دسته‌ها | فروشنده شوید | ورود]

[Hero]
  H1: چی می‌خوای بخری؟ بیزبورد کمک می‌کنه بهتر انتخاب کنی.
  [AI Chat Box بزرگ وسط صفحه]
    placeholder: مثلا بنویس «گوشی تا ۳۰ میلیون با دوربین خوب»
    prompt chips: گوشی، لپ‌تاپ، کولر گازی، تلویزیون، هدفون
  [Classic Search کوچک‌تر]

[Trust strip]
  قیمت به‌روز | فروشنده معتبر | نقد واقعی | خرید امن

[Popular Categories]
  موبایل | لپ‌تاپ | لوازم خانگی | صوتی تصویری | گجت

[Trending Products]
  card product

[Best Sellers]
  seller card

[SEO Intro کوتاه و طبیعی]
[Footer]
```
