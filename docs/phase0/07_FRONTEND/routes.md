# مسیرهای فرانت‌اند MVP

## Public

- `/` صفحه اصلی AI-first
- `/search` جستجوی عمومی
- `/category/[categorySlug]` صفحه دسته
- `/product/[productSlug]` صفحه محصول
- `/compare` مقایسه
- `/b/[businessSlug]` مینی‌سایت کسب‌وکار
- `/shop/[businessSlug]` alias اختیاری برای فروشگاه‌ها
- `/login` ورود
- `/privacy` حریم خصوصی
- `/terms` قوانین استفاده
- `/seller-terms` قوانین فروشنده
