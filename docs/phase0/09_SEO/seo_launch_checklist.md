# چک‌لیست سئو قبل از لانچ MVP

## Technical SEO

- [ ] صفحات اصلی با SSR/ISR رندر می‌شوند.
- [ ] title و meta description برای صفحه اصلی، دسته، محصول و مینی‌سایت وجود دارد.
- [ ] هر صفحه فقط یک H1 دارد.
- [ ] canonical صحیح است.
- [ ] robots.txt درست است.
- [ ] sitemap محصولات/دسته‌ها/کسب‌وکارها ساخته می‌شود.
- [ ] صفحات فیلتر بی‌ارزش noindex هستند.
- [ ] 404/410 درست مدیریت می‌شود.
- [ ] redirectهای مهاجرت از وردپرس آماده‌اند.

## Structured Data

- [ ] BreadcrumbList
- [ ] Product
- [ ] Offer یا AggregateOffer
- [ ] Review/AggregateRating فقط اگر معتبر و واقعی باشد.
- [ ] FAQPage فقط برای FAQ واقعی و مفید.
- [ ] LocalBusiness/Store برای مینی‌سایت‌ها، در صورت داشتن NAP معتبر.

## Content Quality

- [ ] هیچ صفحه AI-generated بدون quality_score و review منتشر نمی‌شود.
- [ ] صفحات دسته فقط لیست محصول نیستند و راهنمای خرید دارند.
- [ ] محتوای کپی‌شده از سایت فروشنده منتشر نمی‌شود مگر مجوز/ارزش افزوده مشخص باشد.
- [ ] عکس‌ها alt text دارند.
- [ ] صفحات thin noindex هستند.

## Performance

- [ ] LCP صفحات اصلی زیر ۲.۵ ثانیه هدف‌گذاری شده است.
- [ ] تصاویر WebP/AVIF یا resize شده‌اند.
- [ ] lazy loading برای تصاویر پایین صفحه فعال است.
- [ ] فونت‌ها بهینه بارگذاری می‌شوند.

## Analytics

- [ ] سرچ‌های بدون نتیجه ثبت می‌شوند.
- [ ] کلیک خروجی فروشنده ثبت می‌شود.
- [ ] تبدیل چت AI به کلیک/خرید ثبت می‌شود.
- [ ] impressionهای SEO landing pages قابل گزارش است.
